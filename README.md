A desktop GUI client to interact with the `[hashes.com](https://hashes.com/)` API and escrow service.

## Usage
```bash
pip install -r requirements.txt
python hashes_gui.py
```

## Notes
- Enter your API key in the top bar and click `Save Key`.
- The app stores local preferences in `gui_settings.json`.
